# Inventory Manager Dashboard

Inventory Manager is a scalable full-stack application designed for efficient inventory tracking and management. It features a modern, responsive frontend built with Next.js, TypeScript, Redux Toolkit, and Tailwind CSS. The backend leverages Node.js, Prisma ORM, and PostgreSQL to provide robust API services.

The app integrates AWS services such as RDS for managed databases, S3 for file storage, EC2 for hosting, Amplify for frontend deployment, and API Gateway for secure API management, supporting seamless cloud scalability and CI/CD workflows.

## Key Features
- Responsive dashboard with advanced data tables (MUI Data Grid) and intuitive sidebar navigation  
- User authentication and role-based access control  
- Product and inventory CRUD operations with real-time updates  
- Expense tracking and reporting  
- Cloud deployment with full AWS integration

## Tech Stack
Next.js, React, TypeScript, Redux Toolkit, Node.js, Prisma ORM, PostgreSQL, AWS (RDS, S3, EC2, Amplify, API Gateway), Tailwind CSS, Material UI
